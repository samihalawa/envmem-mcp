Local MCP Bridge

Keep secrets close to your editor with a local MCP server launched via npx envmem.

{ "mcpServers": { "envmem": { "command": "npx", "args": ["-y", "envmem"], "env": { "ENVMEM_API_KEY": "your-key" } } } }

CLI & MCP Usage

# Option 1: NPX (Recommended)
ENVMEM_API_KEY=your-key npx envmem

# Option 2: Claude Desktop / Cursor / Claude Code config
{
  "mcpServers": {
    "envmem": {
      "command": "npx",
      "args": ["-y", "envmem"],
      "env": {
        "ENVMEM_API_KEY": "your-api-key"
      }
    }
  }
}

# Option 3: Direct URL with API key
{
  "mcpServers": {
    "envmem": {
      "url": "https://envmem.com/mcp?apikey=your-key"
    }
  }
}

# Import your .env file
import_env_variables({
  envText: "OPENAI_API_KEY=sk-...!\nSTRIPE_SECRET_KEY=sk_live_..."
})

# Then search naturally
search_env_variables({ query: "payment processing" })
// Returns: STRIPE_SECRET_KEY, STRIPE_WEBHOOK_SECRET...
